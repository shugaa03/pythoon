Python тіліндегі барлық нәрсе объект ретінде қарастырылады. Төменде Python тіліндегі Oop тұжырымдамаларына қысқаша кіріспе берілген. Класстар мен объекттер - Python класстары объекттің сызбалары болып табылады. Объект – бұл деректерге әсер ететін деректер мен әдістердің жиынтығы. Мұрагерлік - бұл бір класс басқа класстардың қасиеттерін мұраға алатын әдіс. Конструктор - Python конструктор ретінде белгілі __init__() арнайы әдісін ұсынады. Бұл әдіс объектті құру кезінде автоматты түрде шақырылады. Деректер мүшесі - класспен және оның объекттерімен байланысты деректерді сақтайтын айнымалы. Полиморфизм - Полиморфизм - бұл объект көптеген формаларды қабылдай алатын ұғым. Python тілінде полиморфизмге әдісті шамадан тыс жүктеу және әдісті қайта анықтау арқылы қол жеткізуге болады. Әдістің шамадан тыс жүктелуі - Python-да әдісті шамадан тыс жүктеу әдепкі аргументтер арқылы жүзеге асырылады, мұнда әдісті бірнеше параметрлермен анықтауға болады. Әдепкі мәндер әдісті шақыру кезінде кейбір параметрлер берілмесе пайдаланылады. Әдісті қайта анықтау - әдісті қайта анықтау - бұл ішкі класс өзінің суперклассында анықталған әдісті жүзеге асыратын тұжырымдама. Инкапсуляция - инкапсуляция деректер мен әдістерді бір бірлікке орау. Python тілінде инкапсуляцияға жалпы, жеке және қорғалған сияқты қатынас модификаторлары арқылы қол жеткізіледі. Дегенмен, Python қатынас модификаторларын қатаң түрде орындамайды және атау конвенциясы қол жеткізу деңгейін көрсетеді. Деректерді абстракциялау: деректердің күрделілігін жасыру және пайдаланушыға тек маңызды мүмкіндіктерді көрсету әдісі. Ол деректермен өзара әрекеттесу үшін интерфейсті қамтамасыз етеді. Деректерді абстракциялау күрделілікті азайтады және кодты модульді етеді, бұл әзірлеушілерге бағдарламаның маңызды мүмкіндіктеріне назар аударуға мүмкіндік береді.
